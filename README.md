# 🌐 Lok-Lagbe

A web-based task marketplace where anyone can **request help** with everyday tasks, and others can **respond, complete the task, and earn money**. Whether it's house cleaning, tutoring, or tech support — this platform connects people in need with people who can help.

---

## 🚀 Core Idea

The platform allows users to:
- Post tasks they need help with  
- Get matched with helpers based on skills and location  
- Chat, track progress, and complete work  
- Pay securely, with the platform taking a small commission  

---

## 🛠️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/MasudRanaMushfiq/Lok-Lagbe.git
cd Lok-Lagbe

# 2. Install project dependencies
npm install

# 3. Install required packages
npm install firebase
npm install react-native-image-picker
npm install @react-native-picker/picker
npm install @expo/vector-icons

# 4. Start the development server
npx expo start
